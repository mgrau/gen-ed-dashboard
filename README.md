# Gen Ed Dashboard

An interactive tool for exploring how students might interact with General Education models. The dashboard lets you place courses into a semester plan and see in real time how well they satisfy the requirements of each Gen Ed framework under consideration.

## Features

- Browse available Gen Ed courses filtered by requirement category
- Drag-and-drop courses into semester planners
- Track progress across Gen Ed sections with credit counts and progress bars
- Responsive layout for mobile and desktop

## Development

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Deployment

Deployed to GitHub Pages via GitHub Actions on every push to `main`.
